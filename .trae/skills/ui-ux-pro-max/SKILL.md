# UI UX Pro Max

## Purpose
Deliver a higher quality desktop app experience by improving interaction feedback, information hierarchy, and workflow clarity.

## When To Invoke
- User asks for premium/pro-level UI optimization
- Multi-panel pages feel dense or confusing
- Need better action grouping and scanning efficiency

## Working Rules
- Optimize UX with minimal risk: avoid changing backend contracts.
- Group related controls and reduce visual noise.
- Highlight primary vs secondary vs destructive actions.
- Use concise helper text and clear status areas.
- Keep keyboard and mouse usage practical in desktop context.

## UX Checklist
1. Navigation clarity: active state and section orientation.
2. Form usability: labels, field spacing, control consistency.
3. Action hierarchy: primary/secondary/danger semantics.
4. List readability: row density, selected state, meta emphasis.
5. Feedback quality: logs, hints, and empty-state messaging.
6. Responsiveness: avoid overflow and cramped controls.

## Success Criteria
- Users locate key actions faster.
- Dense pages become easier to understand.
- Visual language is consistent across modules.
